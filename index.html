About Me

My name is Tayte Waterman. I'm an engineer, developer, dad, and restless nerd at heart. I love to learn new things, explore new places and perspectives, and share time with friends and family. A Pacific-Northwest Native, I currently live abroad with my wife and young son in Stuttgart, Germany where we enjoy living and working in a new culture.

In my spare time I enjoy tinkering with software and electronics projects, kicking back with a good book or videogame, amateur photography, and travelling with my family to experience new places and things to eat.

Professionally, I have worked for over a decade in the automotive industry working on embedded Mechatronics systems for commercial vehicles. I enjoy working in the intersection of physical systems and software controls; where software IT technologies intersect with real-time, critical controls. I currently work as a Software Architect at Daimler Truck AG, helping define our next generation of software-oriented systems. I am passionate about continual development, both personally, for our organization and industry, and in supporting peers. I believe an engineers role is a hybrid of both technical discipline and creativity. I love learning from new challenges and the diverse perspectives of my colleauges.

Projects

8-Channel MIDI Stepper-Motor Player
Dec 2024
LINK

A joint Raspberry Pi and Arduino based system that reads MIDI files and plays them via stepper motors using the motor frequency to play individual notes. The Raspberry Pi consumes and translates MIDI files into a proprietary set of commands via USB to an Arduino controller responsible for driving the stepper motors themselves. Several example MIDIs and hardware setup details are included.

OpenGL
Nov 2023
LINK

An Open GL project created as part of the CS 547 Computer Graphics at PSU. This project implements a 3D scene with various objects, demonstrating several 3D graphics techniques built on top of Open GL. Features include texture mapping, lighting effects, camera controls, and object transformations. The project showcases the fundamentals of 3D rendering and user interaction within a graphical environment.

Sorting Algorithm Visualization
Dec 2022
LINK LINK

A web-based visualization of various sorting algorithms implemented in JavaScript and HTML/CSS. The project includes animations that demonstrate how different sorting algorithms work, such as Bubble Sort, Quick Sort, Merge Sort, and Insertion Sort. Users can select different algorithms to see step-by-step how they sort an array of numbers, providing an educational tool for understanding sorting techniques.
