About Me

My name is Tayte Waterman. I'm an engineer, developer, dad, and restless nerd at heart. I love to learn new things, explore new places and perspectives, and share time with friends and family. A Pacific-Northwest native, I currently live abroad with my wife and young son in Stuttgart, Germany working for Daimler Truck AG as a Software Architect for commercial vehicle electrical systems. In parallel I am currently enrolled at Portland State University, pursuing a Master's in Computer Science.

In my spare time I enjoy tinkering with software and electronics projects, kicking back with a good book or videogame, amateur photography, and travelling with my family to experience new places and things to eat.

Professionally, I have worked for over a decade in the automotive industry working on embedded Mechatronics systems for commercial vehicles. I enjoy working in the intersection of physical systems and software controls; where software IT technologies intersect with real-time, critical controls. I am passionate about continual development, both personally, for our organization and industry, and in supporting peers. I believe an engineers role is a hybrid of both technical discipline and creativity. I love learning from new challenges and the diverse perspectives of my colleauges.

8-Channel MIDI Stepper-Motor Player
About

In this project is defined joint Raspberry Pi and Arduino code used to play MIDI-format music files over 8 stepper motors. The Raspberry Pi code reads MIDI-format music files and translates them into note commands split across 8-channels. The 8-channels each map to a stepper motor, which "plays" the corresponding note via the frequency used to drive it. Two Arduinos are used, controlling 4-steppers each. This project explores a combined micro-processor and micro-controller architecture, unifying the needs of high-precision real-time controls, and higher abstraction file-system access and processing.

OpenGL Demo
About

In this project I implemented a 3D scene via the C++ OpenGL library for my CS547, Computer Graphics final at PSU. The implemented scene is an outdoor, forest "park" containing several features used to demonstrate different OpenGL features and 3D graphics techniques. In example, there is a central mountain with various levels of proceduraly generated face-detail, animated fyling birds, parametric objects which can vary between instances, and an ability to switch views and change environmental lighting.

Sorting Algorithm Visualization
About

A web-based visualization of various sorting algorithms implemented in JavaScript and HTML/CSS. The project includes animations that demonstrate how different sorting algorithms work, such as Bubble Sort, Quick Sort, Merge Sort, and Insertion Sort. Users can select different algorithms to see step-by-step how they sort an array of numbers, providing an educational tool for understanding sorting techniques.
